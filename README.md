# 待办助手 - Tauri2.0 + Vue3 桌面应用

基于 Tauri 2.0 和 Vue3 开发的桌面待办助手应用，支持 AI 智能解析任务、定时提醒、卡通宠物等功能。

## 功能特性

### 核心功能
- ✅ **AI 智能录入** - 支持文本/语音输入，AI 自动解析任务内容、截止时间、优先级和工作量
- ✅ **任务管理** - 完整的任务 CRUD 操作，支持优先级、工作量、截止时间等属性
- ✅ **今日待办** - 自动筛选和排序今日需要完成的任务
- ✅ **3天到期提醒** - 自动检测即将到期的任务并提醒
- ✅ **卡通宠物提醒** - 每小时弹窗提醒用户起身活动
- ✅ **系统托盘** - 常驻托盘，支持隐藏/显示主窗口
- ✅ **主题切换** - 支持浅色/深色主题
- ✅ **本地存储** - SQLite 本地数据库，数据隐私安全

### 技术栈
- **前端**: Vue3 + Vite + Element Plus + Pinia
- **后端**: Tauri 2.0 + Rust
- **数据库**: SQLite
- **AI**: 支持大模型 API（豆包、通义千问等）

## 快速开始

### 环境要求
- Node.js 18+
- Rust 1.70+
- npm 或 yarn

### 安装依赖

```bash
npm install
```

### 开发模式

```bash
npm run tauri:dev
```

### 构建应用

```bash
npm run tauri:build
```

## 使用说明

### 1. AI 智能录入
- 在「AI 录入」标签页输入任务描述
- 点击「AI 解析」按钮，系统会自动解析任务信息
- 支持自然语言描述，例如：
  - "下周三之前完成需求文档，这个工作比较复杂，尽快做完"
  - "明天下午3点开会，重要任务"

### 2. 配置 API Key
- 点击右上角设置按钮
- 在「API Key」输入框中填入你的大模型 API Key
- 支持豆包、通义千问等兼容 OpenAI 格式的 API

### 3. 宠物提醒设置
- 在设置中可以开启/关闭宠物提醒
- 可自定义提醒间隔（30分钟/1小时/2小时）

### 4. 系统托盘
- 应用启动后会自动最小化到托盘
- 右键托盘图标可以：打开主窗口、隐藏窗口、退出程序
- 点击关闭按钮会隐藏窗口而不是退出程序
